<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">在人工神经网络中，突触权重（Input</span><span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000"> </span><span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">Synaptic</span><span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000"> </span><span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">Weights）、求和结</span>

<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">1.突触权重</span>
<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">在生物神经系统中，突触是神经元之间传递信号的结构。在人工神经</span>
<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">网络中，突触权重模拟了这种传递效率，即一个神经元对另一个神经</span>
<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">元信号的影响程度。</span>
<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">权重是连接神经元的线（或称为突触）上的数值，代表输入信号的重</span>
<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">要性或强度。权重越高，对应的输入信号对神经元的影响越大。</span>
<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">在训练过程中，权重会根据网络的学习算法进行调整，以优化网络的</span>
<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">性能。</span>
<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">2.求和结点</span>
<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">求和结点是神经元内部的一个处理单元，它的作用是将所有输入信号</span>
<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">（乘以相应的权重）进行累加，形成一个总和。这个总和通常还会加</span>
<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">上一个偏置项（bias），偏置项是一个常数，用于调整求和结果的阈</span>
<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">值。</span>
<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">求和结点的输出是输入信号加权后的总和，这个总和将被送入激活函</span>
<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">数中进行下一步处理。</span>

<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">激活函数也称为传递函数，是神经网络中体现非线性能力的关键因</span>

<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">Sigmoid函数能够将输入压缩到0～1，适合用于二分类问题。Tanh函</span>

<span style="font-family:'Unnamed-T3';font-size:16.69pt;color:#000000">编解码架构概述</span>

<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">编解码架构（Encoder-Decoder</span><span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000"> </span><span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">Architecture）是一种常用于处理序</span>

<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">列转换成输出序列的场景，如机器翻译、文本摘要、语音识别等。</span>
<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">编解码架构主要由两部分组成：</span><span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000"> </span>
<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000"> </span><span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">编码器（Encoder）：负责读取输入序列并将其转换成一个固定大小</span>
<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">的内部表示，通常称为上下文向量（Context</span><span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000"> </span><span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">Vector）。该向量捕获</span>
<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">了输入数据的主要特征和语义信息。</span><span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000"> </span>
<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000"> </span><span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">解码器（Decoder）：使用编码器生成的上下文向量来生成输出序</span>
<span style="font-family:'Unnamed-T3';font-size:14.83pt;color:#000000">列。在训练阶段，解码器的目标是生成与目标序列尽可能相似的输</span>
